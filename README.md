# neom-game-character-name
Name Generator

The name generator is hoped to build character names for the game series of Neom. The information is just forming about the
game. The races and cultures are a fun way for players to express themselves or role-play in the game. The first game in the series
that is being worked on is Neom I: Nacirema

The name generator is not yet built and is just in pre-mature form for the game, the name generator will be a fun tool for seeing current
character names along with the possibility of seeing future game series names that might be hidden in some places for the purpose of 
preparing a name that will be current, a player cannot play as a character if the character has not been released in the game yet, so 
this is a perfect opportunity getting involved in the game

The game is hoped for Git Hub to get involved with the game through creating names for the game in Git Hub as a reliable source for the game names in the game series of Neom
